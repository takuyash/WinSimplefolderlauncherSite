Windows向けシンプルフォルダランチャー

WinSimpleFolderLauncher

WinSimpleFolderLauncherは、Windows向けの軽量フォルダランチャーです。
フォルダ構成をそのまま使えるため設定不要で直感的に操作できます。
個人利用・商用利用を問わず、無料で使用することができます。

コンセプト

動作環境

Windows 11 / Windows 10

Features

📁 フォルダ＝ランチャー

フォルダ構成がそのままランチャー。Explorer感覚で操作できます。

⚡ 超高速・軽量

常駐しても負荷は最小限。起動も表示も一瞬です。

⌨ キーボード操作

Shift 2回連続 (キーの種類と回数は設定で変更できます)もしくは Ctrl + Shift + i で即起動。
さらに↑↓移動やショートカットで呼び出せるのでキーボードのみで操作が完結。

🔍 キーワード検索機能

キーワードで絞り込むことができます。

🔧 設定不要

使い始めるのに必要なのはexeを配置してパスを指定するだけ。面倒な設定はありません。

🔧 複数言語対応

日本語と英語に対応しています。

🔧 ホットキーの変更

ホットキー起動のオンオフやキーの種類、連打回数の変更ができます。

How to Use

1. exe をダウンロードして任意の場所に配置

2. ランチャーにしたいフォルダを指定

3. ショートカットキーで即起動

プライバシーについて

本アプリはランチャー起動のホットキー機能のため、キーボード入力イベントを検知します。
入力された文字内容の保存・送信は行いません。
個人情報の収集は行いません。
